# Behavioral Interviewing

## Quick start
```
STAR Framework:
S - Situation: Ask candidate to describe the context
T - Task: What was their specific responsibility?
A - Action: What steps did they take?
R - Result: What was the outcome?
```

## Workflows
1. **Prepare questions**: Map competencies to behavioral questions
2. **Set context**: Explain STAR format to candidate upfront
3. **Probe deeply**: Follow up with "What specifically did YOU do?"
4. **Score responses**: Use rubric for consistency across candidates
5. **Debrief**: Compare notes with interview panel immediately

### Sample Questions
- "Tell me about a time you disagreed with a technical decision"
- "Describe a project that failed. What did you learn?"
- "Give an example of how you handled conflicting priorities"

## Common Mistakes
- **Accepting hypothetical answers**: Redirect to past experiences
- **Not probing**: Accept surface-level responses without digging deeper
- **Leading questions**: Avoid questions that suggest the "right" answer
- **Ignoring red flags**: Document concerns even if candidate recovers
- **Inconsistent scoring**: Use the same rubric for all candidates
